# SpringBoot Example with various libs

This is an example project bringing together a group of popular libraries.

* SpringBoot
* SpringBoot Security
* SpringBoot Cache + EhCache
* SpringBoot JPA
* Swagger
* FlyWay (disabled)
* Apache HttpComponents + Lang3 + CommonsText + CommonsValidator
* Lombok
* Mockito
* RandomBeans
* H2
* Jacoco (UT + IT)
* Release Plugin
* Checkstyle
